# Exploratory_Data_Analysis_with_Python
Exploratory Data Analysis on customer demographic dataset using Python.

This project focuses on exploring and analyzing a customer demographic or car price determining indicators dataset using Python. The goal is to identify patterns, relationships, and insights that can support decision making, marketing strategies, and customer segmentation.

## Project Overview
The notebook walks through the typical EDA workflow: data loading, cleaning, descriptive statistics, visual exploration, and interpretation of findings. The analysis highlights how demographic factors-  Continuous numerical variables:

Length, Width, Curb-weight, Engine-size, Horsepower, City-mpg, Highway-mpg, Wheel-base, Bore.

Categorical variables: Drive-wheels.

## Key Steps in the Analysis
- Imported and inspected the dataset to understand its structure
- Cleaned the dataset by handling missing values and correcting data types
- Generated descriptive statistics to summarize key variables
- Visualized relationships and patterns using plots and charts
- Interpreted insights and summarized meaningful observations
- Anova Test result
